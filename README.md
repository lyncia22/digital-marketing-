# Lyncia Chiguri | Digital Marketing Portfolio

A high-performance, aesthetically pleasing digital marketing portfolio built for **Lyncia Chiguri**. This project showcases expertise in performance marketing, SEO, and strategic growth, tailored for a results-driven junior professional.

## ✨ Features

- **Modern Tech-Chic Aesthetic:** A "girlish" yet professional palette using Midnight Rose and Lavender tones.
- **AI-Powered Case Study Builder:** Integrated Genkit flow to transform raw project results into compelling narratives.
- **Dynamic Portfolio:** Responsive case study detail pages with clear KPI highlighting.
- **Optimized Typography:** Poppins for headlines and Inter for clean, readable body text.

## 🛠️ Tech Stack

- **Framework:** Next.js 15 (App Router)
- **Styling:** Tailwind CSS & ShadCN UI
- **AI Integration:** Firebase Genkit (Gemini 2.5 Flash)
- **Deployment:** Firebase App Hosting
